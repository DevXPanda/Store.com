# вҡҷпёҸ VegFru вҖ” Backend (Convex DB + API Routes)

Convex real-time database + Next.js API routes for auth and orders.

## Convex Setup
```bash
npm install
npx convex login          # authenticate with Convex cloud
npx convex init           # creates your deployment + generates _generated/
npm run dev               # starts Convex dev server (watches convex/ folder)
```

## Database Tables (convex/schema.ts)

| Table | Fields |
|-------|--------|
| `users` | name, email, passwordHash, role (customer/admin/delivery), phone, isActive |
| `products` | name, category, price, originalPrice, unit, emoji, stock, isActive, badge, rating |
| `orders` | userId, customerName, address, status, subtotal, total, paymentMethod, assignedDeliveryBoyId |
| `orderItems` | orderId, productId, productName, quantity, unitPrice, totalPrice |
| `deliveryTracking` | orderId, deliveryBoyId, status, note, lat, lng, timestamp |

## Order Status Flow
```
pending вҶ’ confirmed вҶ’ preparing вҶ’ assigned вҶ’ picked_up вҶ’ out_for_delivery вҶ’ delivered
                                                                           вҶҳ cancelled
```

## API Routes

### POST /api/auth/login
```json
{ "email": "admin@vegfru.com", "password": "admin123" }
вҶ’ { "success": true, "token": "jwt...", "user": { "id", "name", "email", "role" } }
```

### POST /api/auth/register
```json
{ "name": "Priya", "email": "priya@example.com", "password": "pass123", "phone": "9876543210" }
вҶ’ { "success": true, "userId": "..." }
```

### GET /api/orders?status=pending
```json
вҶ’ { "success": true, "orders": [...] }
```

### POST /api/orders
```json
{
  "userId": "...", "customerName": "...", "customerPhone": "...",
  "deliveryAddress": "...", "subtotal": 328, "total": 328,
  "paymentMethod": "cod", "items": [...]
}
вҶ’ { "success": true, "orderId": "..." }
```

## Convex Mutations (called client-side via Convex SDK)

| Function | Description |
|----------|-------------|
| `products.seedProducts` | Seed 8 demo products |
| `auth.seedAdminAndDelivery` | Create admin + delivery demo accounts |
| `orders.updateOrderStatus` | Change order status |
| `orders.assignDeliveryBoy` | Assign delivery partner to order |
| `orders.getAdminStats` | Dashboard stats (revenue, pending, etc.) |
| `orders.getDeliveryStats` | Delivery boy stats |
